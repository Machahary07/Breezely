# Breezely AI

Breezely AI is an AI-powered browser automation platform that lets you turn simple prompts into powerful, real-world actions on the web effortlessly. With its smart extension and integrated web studio, users can generate, edit, and run automation scripts (like form filling, repetitive tasks, or multi-step workflows) as smoothly as a breeze. Whether using their own API keys for free or upgrading for seamless access, Breezely makes complex browser interactions feel natural, fast, and intuitive, bringing the power of intelligent automation to everyone, without the usual friction.

## Key Features
- **Prompt-to-Action**: Convert natural language into complex browser workflows.
- **Smart Extension**: Control and automate any website directly from your browser.
- **Integrated Web Studio**: Generate, edit, and manage your scripts in a single dashboard.
- **Flexible Access**: Bring your own API keys or use our managed service.

## Tech Stack

### Web Application
- **Frontend**: [Next.js](https://nextjs.org/) (TypeScript + Sass)
- **Backend/DB**: [Supabase](https://supabase.com/)
- **Automation Engine**: [Playwright](https://playwright.dev/)
- **Email Services**: [Resend](https://resend.com/)
- **Hosting**: [Vercel](https://vercel.com/)

### Browser Extension
- **Manifest**: Chrome Extension Manifest V3
- **Core**: Vanilla JavaScript
- **Extension Backend**: Node.js/Express.js
